Rutgers Creative Writing Club Website


I developed a static website using HTML and CSS for the Rutgers Creative Writing Club. The function of this website is to:

• Announce meeting information
• State the organization's purpose
• Inform students of writing-related events hosted by the university and other student organizations
• Provide contact information and links to social media
• Maintain a list of writing resources
